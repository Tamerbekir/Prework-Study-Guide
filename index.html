HTML

  • It is a hyperlink markup language
  • It is the structure to the content
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • In the preceding statement, a element creates the connection to the external file.
  • The href attribute specifies the location (URL) of the external resource. The href attribute is using a relative path. This means the location is derived from the current location (in this case, index.html). To locate the style sheet from the current location, we must look in the assets folder, hence assets/style.css.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git clone (paste SSH URL) creates a clone folder name. All things in the git folder will show up in Code VS.
  • git status : checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout -b feature/(what it is) : checks out in the branch
  • git add -A or git add . : pushes through all changes
  • git commit -m "say what you changed" : pushes though changes with notes attached
  • git pull origin main : Make sure it is in sync with github
  • git push origin "name of branch" : pushes through all changes

    • Pull Request, In Short:
  • 1. Open terminal
  • 2. Open foldering using cd
  • 3. git checkout (name of branch)
  • 4. git Status
  • 5. git add -A
  • 6. git commit -m "(add message)"
  • 7. git pull origin main
  • 8. git push origin "name of branch"

    • See more on the git cheat sheet!

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
